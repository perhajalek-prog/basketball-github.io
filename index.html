ABOUT NBA UNIVERSE

NBA Universe is a basketball themed website created using HTML, CSS and JavaScript. The goal of the website is to give basketball fans a place where they can explore NBA teams, players, and team rosters in a modern and interactive way.

This project includes animated team cards, responsive layouts, player roster sections, smooth scrolling navigation and a professional sports design inspired by official NBA websites.

Users can browse different NBA franchises such as the Lakers, Warriors, Bulls, Celtics, Heat and Bucks while viewing some of their most important players and positions.

The website was designed to work on both desktop and mobile devices, making it easy to navigate from anywhere. More features can later be added including live NBA scores, standings, statistics, dark mode, player search systems, highlight videos and advanced animations.

NBA Universe is perfect as a beginner or intermediate web development project because it combines design, layout, navigation and JavaScript interactions into one complete basketball website experience.
